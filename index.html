Guess The Word

The game is simple, type a word and it will display colors to each character.

▣ Green means the character is in the right index of the word.

▣ Yellow means the word contains the character, but it's in a different index.

▣ Red means there's no such character in the word!

There are unlimited tries in this game and you can also set your own answer word!

The answer can be edited on the gameLogic.js file inside resources/js.

The answer can also be found inspecting this page's console logs!

The SUBMIT button checks if the answer is right and displays the colors indicating right/misplaced/wrong characters.

The CLEAR button clears the word input, allowing to use the keyboard, but you can also click on each input and type the characters(I personally prefer this way).
